# AlphaLens AI

AlphaLens AI is an AI-powered crypto intelligence platform built on Mantle.

The platform allows users to analyze crypto assets such as BTC, ETH, and MNT using AI-generated market insights and permanently store those insights on the Mantle blockchain for transparency and verification.

## Problem

Crypto investors often rely on fragmented information sources and unstructured market opinions. Important insights are difficult to verify and frequently disappear over time.

## Solution

AlphaLens AI combines AI-powered market analysis with on-chain storage.

Users can:

* Connect their wallet using MetaMask
* Analyze crypto assets using Gemini AI
* Receive structured market intelligence
* Store AI-generated analysis on Mantle blockchain
* Create a transparent and verifiable record of insights

## Features

* AI-powered token analysis
* Alpha Score generation
* Risk assessment
