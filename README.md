# AlphaLens AI

AlphaLens AI is an AI-powered crypto analysis platform built on Mantle.

Users can enter a token such as BTC, ETH, or MNT and receive:

* Alpha Score
* Risk Assessment
* Bullish Factors
* Bearish Factors
* Investment Recommendation

The analysis can be permanently recorded on Mantle using a deployed smart contract.

## Features

* Gemini AI analysis
* MetaMask wallet integration
* Mantle blockchain integration
* On-chain storage of alpha scores
* User-friendly interface

## Tech Stack

* Node.js
* Express
* Gemini API
* Ethers.js
* Solidity
* Mantle Sepolia
* MetaMask
